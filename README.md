# IPL Player Availability

An event-driven player availability pipeline that collects official cricket news, extracts injuries, recoveries, suspensions, and replacements into structured historical events for AI-powered IPL match prediction.

## Features

- Modular architecture
- IPL Official & ESPN Cricinfo collectors
- Deterministic rule-based parser
- Event-driven availability tracking
- SQLite/PostgreSQL compatible
- Comprehensive test suite
